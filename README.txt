Install required dependency using "pip -q install -r requirements".

The following ulasan_aplikasi.csv was kept as a reference. Feel
free to remove it. But in order for another scraping dataset to be
made, ulasan_aplikasi.csv must be removed beforehand.

These files must be run in the following order:
1. scraping_dataset.py
2. notebook.ipynb

scraping_dataset.py will create two csv files.
1. ulasan_aplikasi.csv
2. clean_data.csv
